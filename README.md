# DevConsole

![Status](https://img.shields.io/badge/Status-Live-success) ![Tools](https://img.shields.io/badge/Catalog-53%20links-2563eb) ![Site](https://img.shields.io/badge/Web-www.devconsole.org-0ea5e9)

DevConsole is a professional, browser-based developer workspace that combines API testing, Coder AI, database query building, one-time secret sharing, and a broad utility toolkit in one platform.

## Live App

- https://www.devconsole.org

## Core Highlights

- 50+ utility experience (docs catalog includes 53 links)
- API Tester with REST/GraphQL, auth, test assertions, and code snippets
- Coder AI for coding assistance and exportable chat history
- DB Query Builder for SQL, MongoDB, Redis, Cassandra, Elasticsearch
- One-Time Secrets for secure single-view sharing
- Docs tools with OpenAPI viewer + API doc generator
