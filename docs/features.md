## Platform Scope

- **Main URL:** https://www.devconsole.org
- **Primary tabs:** Home, Dev Tools, Coder AI, API Testing, DB Query Builder, One-Time Secrets, Blogs
- **Catalog coverage in this doc:** 53 tool links (51 utility entries + 2 core Dev Tools tabs)

## High-Priority Features

### Coder AI

- **Tab:** https://www.devconsole.org?tab=coder-ai
- Chat-based coding assistant
- Save/load history, regenerate/edit prompts, export chat to Markdown/JSON
- Code block highlighting with copy/run actions for supported languages

### API Tester

- **Tab:** https://www.devconsole.org?tab=api-testing
- REST + GraphQL testing
- Auth methods: Bearer, Basic, API key
- Headers/query/cookies/body modes, env variables, assertions/tests, performance mode
- Import: Postman/cURL/OpenAPI
- Generate snippets in JavaScript, Node, Python, cURL, PHP, Java, Go, C#, Ruby

### One-Time Secrets

- **Tab:** https://www.devconsole.org?tab=one-time-secrets
- Create secure secret links intended for single-view sharing
- Fast flow for temporary credentials and sensitive text handoff
