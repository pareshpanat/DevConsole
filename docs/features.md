# Feature Overview

This page summarizes DevConsole features from the repository and live production site.

## Platform Scope

- Main URL: `https://www.devconsole.org`
- Main sections: Home, Dev Tools, Coder AI, API Testing, DB Query Builder, One-Time Secrets, Blogs
- Utility count:
  - Marketing/metadata: "50+"
  - Current repo utility list: 53

## Dev Tools (Utilities)

Developer Utilities include encoders, converters, formatters, security tools, text/data tooling, and generators.

### Popular utility examples

- JSON Formatter
- JWT Decoder + JWT Generator
- Base64 Tool
- Regex Tester
- SQL Formatter
- XML Formatter
- YAML Converter
- Image Compressor
- cURL Builder
- QR Code Generator
- Password Generator + Strength Meter
- CSV/JSON/XML Converter
- Data Cleaner, Data Profiler, Schema Designer

## Coder AI

- Chat-based coding assistant
- Prompt suggestions
- Code block highlighting
- Copy/run code actions (where supported)
- Conversation save/load/delete
- Export conversation (Markdown/JSON)

## One-Time Secrets

- Create secure one-time secret links
- Share link for single-view retrieval
- Useful for temporary credential sharing

## API Testing

Key capabilities:

- REST + GraphQL request support
- Methods: GET/POST/PUT/PATCH/DELETE
- Auth: Bearer, Basic, API Key
- Headers, query params, cookies, body modes
- Environment variables
- Request tests/assertions
- Performance test mode
- Response views: pretty/raw/tree
- Request import (Postman, cURL, OpenAPI)
- Code snippet generation (JavaScript, Node, Python, cURL, PHP, Java, Go, C#, Ruby)
